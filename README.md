# batch-text-converter
Small Batch Text Converter / Encoder

Small and free batch text converter. This programm change file(s) in folder (and subfolder) encoding (e.g. Windows 1251 --> UTF8)

## Key Features

— Batch codepage change for text files

— Creating catalog tree for files in other place

— Extended encoding list

## v 0.0.2b update:

- Add rewrite option in target catalog

- Portable mode by default

- Config file format changed by XML

## System requirements:

Windows XP/7/8/10

.NET Framework 2.0.

256 Mb memory

## Command line keys:

/help - help (on russian)

/np - disable Portable mode

(user settings saved in C:\Users\<username>\AppData\Local\BatchTextConverter\)
